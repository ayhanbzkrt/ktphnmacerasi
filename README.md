# Kütüphane Macerası / Library Adventure

**Kütüphaneyi oynayarak keşfet.**  
**Discover the library by playing.**

Kütüphane Macerası; çocuklar ve gençler için tasarlanmış, kütüphane temalı, tek oyunculu, etkileşimli bir HTML5 oyun deneyimidir. Oyun; keşif, öğrenme, görev tamamlama, rehber karakterlerle etkileşim ve bilgi odaklı mini oyun hissini bir araya getirir. Masaüstü, mobil cihaz ve kiosk/dokunmatik ekran kullanımı düşünülerek geliştirilmiştir.

**Kütüphane Macerası** is a single-player, interactive HTML5 game experience designed for children and young learners. It combines exploration, learning, mission-based progression, guide characters, and knowledge-themed gameplay in a magical library setting. It is designed with desktop, mobile, and kiosk/touchscreen use in mind.

## Oynanış / Gameplay

### Türkçe
Oyuncu, büyülü bir kütüphane dünyasında ilerlerken görevleri tamamlar, nesneleri toplar, rehber karakterlerle konuşur ve farklı bilgi odaklı etkileşimlerle karşılaşır. Oyun yapısı; keşif, toplama, etkileşim ve ilerleme döngüsü üzerine kuruludur.

### English

## Kontroller / Controls

### Türkçe
**Klavye**
- **Yön Tuşları**: Hareket
- **Boşluk (Space)**: Zıpla
- **Shift**: Koş / Dash
- **F**: Ateş / Saldırı
- **E**: Konuş / Etkileşim
- **Esc / Menü**: Duraklat veya menüyü aç

**Fare / Dokunmatik**
- Menü butonları ile seçim yapılabilir
- Mobil ve kiosk kullanımında ekran üstü kontrol butonları kullanılabilir

### English
**Keyboard**
- **Arrow Keys**: Move
- **Space**: Jump
- **Shift**: Dash
- **F**: Fire / Attack
- **E**: Talk / Interact
- **Esc / Menu**: Pause or open menu

**Mouse / Touch**
- Menu buttons can be used for navigation and selection
- On mobile and kiosk devices, on-screen controls can be used
